Distribution Entropy: DistEn()

Table of Contents
Syntax: References: Note: Example 1: Example 2: Example 3: Example 4:

Syntax:

[Dist, Ppi] = DistEn(Sig, 'm', 2, 'tau', 1, 'Bins', 'sturges', 'Logx', 2, 'Norm', true)
  • Sig Time series signal, a vector of length > 10.
  • m Embedding dimension, a positive integer.
  • tau Time delay, a positive integer.
  • Bins Histogram bin selection method, in integer > 1 indicating the number of bins or one of the following strings: "sturges", "sqrt", "rice", "doanes" [default: "sturges"]
  • Logx Logarithm base in Shannon's entropy formula, a positive scalar. (Enter 0 for natural logarithm)
  • Norm Normalization of Dist value: false - no normalisation / true - normalises w.r.t number of histogram bins (default)
  • Dist Distribution entropy estimate.
  • Phi Probability of each histogram bin.

References:

  1. Li, Peng, et al., Assessing the complexity of short-term heartbeat interval series by distribution entropy, Medical & Biological Engineering & Computing 53.1 (2015): 77-87.

Note:

In the following examples, example signals / data are imported into MatLab using the ExampleData function from EntropyHub.
To use this function as outlined in the examples below, an internet connection is required.
ExampleData accepts any of the following strings:
  • 'uniform' uniformly distributed random numbers in range [0 1]
  • 'gaussian' normally distributed random numbers with mean = 0, and standard deviation = 1
  • 'randintegers' uniformly distributed pseudorandom integers in range [1 8]
  • 'chirp' chirp signal with the following parameters: f0 = .01, t1 = 4000, f1 = .025
  • 'lorenz' X, Y, Z components of the Lorenz system (sigma: 10, beta: 8/3, rho: 28) [Xo = 10, Yo = 20, Zo = 10]
  • 'henon' X, Y components of the Henon attractor (alpha: 1.4, beta: .3) [Xo = 0, Yo = 0]

Example 1:

Import a signal of uniformly distributed random numbers in the range [0,1] and calculate the distribution entropy for an embedding dimension (m) of 2.
X = ExampleData('uniform');
DistEn(X)
ans = 0.9587

Example 2:

Import a signal of pseudorandom integers in the range [1, 8] and calculate the distribution entropy with an embedding dimension of 5, a time delay (tau) of 3, and Sturges' bin selection method.
X = ExampleData('randintegers');
Dist = DistEn(X, 'm', 5, 'tau', 3)
Note: 16/24 bins were empty
Dist = 0.5318
Change the number of histogram bins using Rice's method.
Dist = DistEn(X, 'm', 5, 'tau', 3, 'Bins', 'rice')
Note: 398/406 bins were empty
Dist = 0.2814

Example 3:

Calculate the distribution entropy of a chirp signal using an embedding dimension of 6, a time delay of 4, Doane's binning method, and normalise the result w.r.t. the number of histogram bins.
X = ExampleData('chirp');
Dist = DistEn(X, 'm', 6, 'tau', 4, 'Bins', 'doanes', 'Norm', true)
Dist = 0.9470

Example 4:

Import the x and y components of the Henon system of equations.
Data = ExampleData('henon');
figure('Color', 'k')
plot(Data(:,1), Data(:,2), 'g.')
xlabel('x-component','color','g'), ylabel('y-component','color','g')
set(gca,'color','k'), axis square
Calculate the distribution entropy of the x-component in nats (natural logarithm) and return the probabilties of each histogram bin (Ppi) with all other default parameter values.
X = Data(:,1);
[Dist, Ppi] = DistEn(X, 'Logx', 0)
