Increment Entropy: IncrEn()

Table of Contents
Syntax: Note: Example 1: Example 2: Example 3:

Syntax:

Incr = IncrEn(Sig, 'm', 2, 'tau', 1, 'R', 4, 'Logx', exp(1), 'Norm', false)
  • Sig Time series signal, a vector of length > 10.
  • m Embedding dimension, an integer > 1.
  • tau Time delay, a positive integer.
  • R Quantifying resolution, a positive scalar.
  • Logx Logarithm base in Shannon's entropy formula, a positive scalar.
  • Norm Normalisation of Incr value: false - no normalisation (default) / true normalises w.r.t. embedding dimension (m-1)
  • Incr Increment entropy estimate.
References:
  1. Xiaofeng Liu, et al., Increment entropy as a measure of complexity for time series, Entropy 18.1 (2016): 22.1.
  2. *** Correction on Liu, X.; Jiang, A.; Xu, N.; Xue, J. Increment Entropy as a Measure of Complexity for Time Series, Entropy 2016, 18, 22, Entropy 18.4 (2016): 133.
  3. Xiaofeng Liu, et al., Appropriate use of the increment entropy for electrophysiological time series, Computers in biology and medicine 95 (2018): 13-23.

Note:

In the following examples, example signals / data are imported into MatLab using the ExampleData function from EntropyHub.
To use this function as outlined in the examples below, an internet connection is required.
ExampleData accepts any of the following strings:
  • 'uniform' uniformly distributed random numbers in range [0 1]
  • 'gaussian' normally distributed random numbers with mean = 0, and standard deviation = 1
  • 'randintegers' uniformly distributed pseudorandom integers in range [1 8]
  • 'chirp' chirp signal with the following parameters: f0 = .01, t1 = 4000, f1 = .025
  • 'lorenz' X, Y, Z components of the Lorenz system (sigma: 10, beta: 8/3, rho: 28) [Xo = 10, Yo = 20, Zo = 10]
  • 'henon' X, Y components of the Henon attractor (alpha: 1.4, beta: .3) [Xo = 0, Yo = 0]

Example 1:

Import a signal of uniformly distributed pseudorandom integers in the range [1, 8] and calculate the increment entropy with an embedding dimension (m) of 2 and quantifying resolution (R) of 4.
X = ExampleData('randintegers');
IncrEn(X)
ans = 4.6439

Example 2:

Import a chirp signal.
X = ExampleData('chirp');
Calculate the increment entropy with embedding dimension of 5, time delay (tau) of 3, and a quanitfying resolution (R) equal to 3.
Incr = IncrEn(X, 'm', 5, 'tau', 3, 'R', 3)
Incr = 4.8957

Example 3:

Import the x, y and z components of the Lorenz system of equations.
Data = ExampleData('lorenz');
figure('Color', 'k')
plot3(Data(:,1), Data(:,2), Data(:,3), 'g.')
xlabel('x-component','color','g'),
ylabel('y-component','color','g'),
zlabel('z-component','color','g'),
view(-10,10), set(gca,'color','k'), axis square
Calculate the increment entropy of the x component in bits (logarithm base 2), with an embedding dimension of 6, and normalize the result w.r.t. the embedding dimension.
X = Data(:,1);
Incr = IncrEn(X, 'm', 6, 'Logx', 2, 'Norm', true)
Incr = 0.7540
