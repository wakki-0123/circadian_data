Syntax:

[SE2D, Phi1, Phi2] = SampEn2D(Mat, 'm', floor(size(Mat)/10), 'tau', 1, 'r', 0.2*std(Mat), 'Logx', exp(1), 'Lock', true)
  • Mat N x M matrix, where N, M > 10.
  • m Embedding dimension, [default: (floor(N/10) floor(M/10))] - an integer > 1 for square submatrix embedding, or a two-element vector of integers > 1 representing the height and width of the template submatrix.
  • tau Time delay, a positive integer.
  • r Distance threshold value, a positive scalar.
  • Logx Logarithm base in Shannon's entropy formula, a positive scalar.
  • Lock See note on matrix size locking true Matrix height (N) and width (M)
  • SE2D Bidimensional sample entropy estimate.
  • Phi1 The number of matched submatrices for embedding dimensions (m).
  • Phi2 The number of matched submatrices for embedding dimensions (m+1).

About SampEn2D():

SampEn2D has an important keyword argument: Lock. Bidimensional entropy functions are "locked" by default (Lock == true) to only permit matrices with a maximum size of 128 x 128. The reason for this is because there are hundreds of millions of pairwise calculations performed in the estimation of bidimensional entropy, so memory errors often occur when storing data on RAM. e.g. For a matrix of size [200 x 200], an embedding dimension (m) = 3, and a time delay (tau) = 1, there are 753,049,836 pairwise matrix comparisons (6,777,448,524 elemental subtractions). To pass matrices with sizes greater than [128 x 128], set 'Lock' to false.
*** WARNING: unlocking the permitted matrix size may cause MatLab to crash.***

Note:

In the following examples, sample matrices are imported into MatLab using the ExampleData function from EntropyHub.
To use this function as outlined in the examples below, an internet connection is required.
ExampleData accepts any of the following strings:
  • 'uniform_Mat' uniformly distributed random numbers in range [0 1]
  • 'gaussian_Mat' normally distributed random numbers with mean = 0, and standard deviation = 1
  • 'randintegers_Mat' uniformly distributed pseudorandom integers in range [1 8]
  • 'mandelbrot_Mat' image of fractal generated from the mandelbrot set
  • 'entropyhub_Mat' image of the entropyhub logo

Example 1:

Import a matrix of normally distributed random numbers.
X = ExampleData('gaussian_Mat');
figure('Color','k'),
imshow(X,'InitialMagnification',500),
cmap = colormap('turbo'); cmap(:,1) = 0;
colormap(cmap)
Calculate the bidimensional sample entropy using an template matrix (m) with dimensions , where N and M are the number of rows and columns of the input matrix (X), and a distance threshold value equal to twice the matrix standard deviation ().
SE2D = SampEn2D(X, 'r', 2*std(X(:)))
SE2D = 3.2809
Change the size of the template matrix to [6 x 6].
SE2D = SampEn2D(X, 'm', 6, 'r', 2*std(X(:)))
SE2D = 2.1950

Example 2:

Import a matrix of uniformly distributed pseudorandom integers in the range [1, 8] and calculate the bidimensional sample entropy with a template matrix of size [3 x 4], a time delay (tau) of 3 and a distance threshold of 3.
X = ExampleData('randintegers_Mat');
figure('Color','k'),
imshow(X,[],'InitialMagnification',500),
cmap = colormap('turbo'); cmap(:,1) = 0;
colormap(cmap)
SE2D = SampEn2D(X, 'm', [3 4], 'tau', 3, 'r', 3)
SE2D = 5.1085

Example 3:

Import an image of a Mandelbrot fractal as a matrix.
X = ExampleData('mandelbrot_Mat');
figure('Color','k'),
imshow(X,[],'InitialMagnification',500),
colormap('hot')
Calculate the bidimensional sample entropy with a template matrix of size [4 x 4], a time delay (tau) of 2 and return the log-sum of matched sub-matrices for embedding dimensions m (Phi1) and m+1 (Phi2).
[SE2D, Phi1, Phi2] = SampEn2D(X, 'm', 4, 'tau', 2)
SE2D = 2.2282
Phi1 = 2.4364e-05
Phi2 = 2.6246e-06

Example 4:

Import an image of the EntropyHub logo as a matrix and calculate the bidimensional sample entropy in dits (logarithm base 10) and the log-sum of matched sub-matrices using a template matrix of size [6 x 3].
X = ExampleData('entropyhub_Mat');
figure('Color','k'),
imshow(X,[],'InitialMagnification',500),
colormap('cool')
[SE2D, Phi1, Phi2] = SampEn2D(X, 'm', [6 3], 'Logx', 10)
SE2D = 0.1479
Phi1 = 0.1870
Phi2 = 0.1330
