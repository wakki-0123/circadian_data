Syntax:

Data = ExampleData(SigName)
  • SigName A case-sensitive string corresponding to a valid dataset name (see list below).

About ExampleData():

Returns sample datasets (time series or matrices) with specific properties that are commonly used as benchmarks for assessing the performance of various entropy methods. The datasets returned by the ExampleData function are used in the examples provided in documentation on www.EntropyHub.xyz and elsewhere.
Note: The ExampleData function requires an internet connection to download and import the required datasets!
ExampleData imports datasets with any of the following strings:
  • 'uniform' {[N,1] vector} uniformly distributed random numbers in range [0 1]
  • 'gaussian' {[N,1] vector} normally distributed random numbers with mean = 0, and standard deviation = 1
  • 'randintegers' {[N,1] vector} uniformly distributed pseudorandom integers in range [1 8]
  • 'chirp' {[N,1] vector} chirp signal with the following parameters: f0 = .01, t1 = 4000, f1 = .025
  • 'lorenz' {[N,3] matrix} X, Y, Z components of the Lorenz system (sigma: 10, beta: 8/3, rho: 28) [Xo = 10, Yo = 20, Zo = 10]
  • 'henon' {[N,2] matrix} X, Y components of the Henon attractor (alpha: 1.4, beta: .3) [Xo = 0, Yo = 0]
  • 'uniform2' {[N,2] matrix} uniformly distributed random numbers in range [0 1]
  • 'gaussian2' {[N,2] matrix} normally distributed random numbers with mean = 0, and standard deviation = 1
  • 'randintegers2' {[N,2] matrix} uniformly distributed pseudorandom integers in range [1 8]
  • 'uniform_Mat' {[50, 50] matrix} uniformly distributed random numbers in range [0 1]
  • 'gaussian_Mat' {[60, 120] matrix} normally distributed random numbers with mean = 0, and standard deviation = 1
  • 'randintegers_Mat' {[88, 88] matrix} uniformly distributed pseudorandom integers in range [1 8]
  • 'entropyhub_Mat' {[127, 95] matrix} image of the EntropyHub logo
  • 'mandelbrot_Mat' {[127, 95] matrix} image of the Mandelbort fractal

Example 1:

Import and plot the dataset of Gaussian-distibuted random numbers.
Data = ExampleData('gaussian');
figure()
position1 = [.1 .3 .5 .4];
subplot('Position',position1)
plot(Data)
xlabel('Sample number (#)', 'Fontweight', 'bold', 'FontSize',15),
ylabel('Amplitude (a.u.)', 'Fontweight', 'bold', 'FontSize',15),
position2 = [.7 .35 .25 .35];
subplot('Position',position2)
histogram(Data, 20);
xlabel('Sample value', 'Fontweight', 'bold', 'FontSize',15),
ylabel('Frequency', 'Fontweight', 'bold', 'FontSize',15),
 

Example 2:

Import the x and y components of the Henon system of equations.
Data = ExampleData('henon');
figure()
position1 = [.1 .1 .4 .3];
subplot('Position',position1)
plot(Data(1:500,1))
xlabel('Sample number (#)', 'Fontweight', 'bold', 'FontSize',10),
ylabel('Amplitude (a.u.)', 'Fontweight', 'bold', 'FontSize',10),
title('x-component', 'Fontweight', 'bold', 'FontSize',10),
position2 = [.1 .6 .4 .3];
subplot('Position',position2)
plot(Data(1:500,2))
xlabel('Sample number (#)', 'Fontweight', 'bold', 'FontSize',10),
ylabel('Amplitude (a.u.)', 'Fontweight', 'bold', 'FontSize',10),
title('y-component', 'Fontweight', 'bold', 'FontSize',10),
position3 = [.6 .25 .35 .5];
subplot('Position',position3)
plot(Data(:,1), Data(:,2), 'g.')
xlabel('x-component','Fontweight', 'bold', 'FontSize',10),
ylabel('y-component','Fontweight', 'bold', 'FontSize',10),
set(gca,'color','k')

Example 3:

Import a two column matrix of randomly distributed integers and plot each series separately.
Data = ExampleData('randintegers2');
figure()
position1 = [.1 .1 .45 .3];
subplot('Position',position1)
plot(Data(1:1000,1))
xlabel('Sample number (#)', 'Fontweight', 'bold', 'FontSize',10),
ylabel('Amplitude (a.u.)', 'Fontweight', 'bold', 'FontSize',10),
title('Sequence #1', 'Fontweight', 'bold', 'FontSize',10),
position2 = [.1 .6 .45 .3];
subplot('Position',position2)
plot(Data(1:1000,2))
xlabel('Sample number (#)', 'Fontweight', 'bold', 'FontSize',10),
ylabel('Amplitude (a.u.)', 'Fontweight', 'bold', 'FontSize',10),
title('Sequence #2', 'Fontweight', 'bold', 'FontSize',10),
position3 = [.7 .1 .25 .3];
subplot('Position',position3)
histogram(Data(:,1),length(unique(Data(:,1))))
xlabel('Sample value', 'Fontweight', 'bold', 'FontSize',10),
ylabel('Frequency', 'Fontweight', 'bold', 'FontSize',10),
position4 = [.7 .6 .25 .3];
subplot('Position',position4)

Example 4:

Import an image of a Mandelbrot fractal as a matrix and plot the result.
X = ExampleData('mandelbrot_Mat');
 
figure(),
imshow(X,[],'InitialMagnification',200),
colormap('hot')

Example 5:

Import an image of the EntropyHub logo as a matrix and plot the result with a unique colorscheme.
X = ExampleData('entropyhub_Mat');
 
map = [0.3*(2:256)', .7*(256:-1:2)', 0.9*(256:-1:2)']/256;
map = [ones(1,3); map];
 
figure()
imshow(256*(X-min(X))/range(X(:)), map,'InitialMagnification',200)
