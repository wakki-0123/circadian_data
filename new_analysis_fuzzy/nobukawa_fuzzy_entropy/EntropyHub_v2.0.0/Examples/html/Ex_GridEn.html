Gridded Distribution Entropy: GridEn()

Table of Contents
Syntax: References: Note: Example 1: Example 2: Example 3: Example 4:

Syntax:

[GDE, GDR, PIx, GIx, SIx, AIx] = GridEn(Sig, 'm', 3, 'tau', 1, 'Logx', exp(1), 'Plotx', false)
  • Sig Time series signal, a vector of length > 10.
  • m Embedding dimension, a positive integer > 1.
  • tau Time delay, a positive integer.
  • Logx Logarithm base in Shannon's entropy formula, a positive scalar.
  • Plotx When Plotx == true, returns Poicaré plot and a bivariate histogram of the grid point distribution (default: false)
  • GDE Gridded distribution entropy estimate.
  • GDR Gridded distribution rate.
  • PIx Percentage of points below the line of identity (LI).
  • GIx Proportion of point distances above the LI.
  • SIx Ratio of phase angles (w.r.t. LI) of the points above the LI.
  • AIx Ratio of the cumulative area of sectors of points above the LI.

Note:

In the following examples, example signals / data are imported into MatLab using the ExampleData function from EntropyHub.
To use this function as outlined in the examples below, an internet connection is required.
ExampleData passes any of the following strings:
  • 'uniform' uniformly distributed random numbers in range [0 1]
  • 'gaussian' normally distributed random numbers with mean = 0, and standard deviation = 1
  • 'randintegers' uniformly distributed pseudorandom integers in range [1 8]
  • 'chirp' chirp signal with the following parameters: f0 = .01, t1 = 4000, f1 = .025
  • 'lorenz' X, Y, Z components of the Lorenz system (sigma: 10, beta: 8/3, rho: 28) [Xo = 10, Yo = 20, Zo = 10]
  • 'henon' X, Y components of the Henon attractor (alpha: 1.4, beta: .3) [Xo = 0, Yo = 0]

Example 1:

Import a signal of uniformly distributed random numbers in the range [0, 1] and calculate the gridded distribution entropy with an embedding dimension of 3.
X = ExampleData('uniform');
GDE = GridEn(X)
GDE = 2.1953
Plot the Poincaré plot and a bivariate histogram of the grid point distribution with the output.
**Note: The colormap of the bivariate histogram is scaled relative to the number of grid points.
GDE = GridEn(X, 'Plotx', true)
GDE = 2.1953

Example 2:

Import a chirp signal and calculate the gridded distribution entropy and gridded distribution rate (GDR) with an embedding dimension of 7, a time delay (tau) of 4, plotting the Poincaré plot and the bivariate histogram of the grid point distribution.
X = ExampleData('chirp');
[GDE , GDR] = GridEn(X, 'm', 7, 'tau', 4, 'Plotx', true)
GDE = 2.9592
GDR = 0.7551

Example 3:

Import a signal of normally distributed random numbers .
X = ExampleData('gaussian');
Calculate the gridded distribution entropy and gridded distribution rate (GDR) in dits (logarithm base 10) with an embedding dimension of 15, plotting the Poincaré plot and the bivariate histogram of the grid point distribution.
[GDE , GDR] = GridEn(X, 'm', 15, 'Logx', 10, 'Plotx', true)
GDE = 1.8477
GDR = 0.6800

Example 4:

Import the x, y and z components of the Lorenz system of equations.
Data = ExampleData('lorenz');
figure('Color', 'k')
plot3(Data(:,1), Data(:,2), Data(:,3), 'g.')
xlabel('x-component','color','g'),
ylabel('y-component','color','g'),
zlabel('z-component','color','g'),
view(-10,10), set(gca,'color','k'), axis square
Calculate the gridded distribution entropy of the x-component in bits (logarithm base 2) using an embedding dimension of 9 and a time delay of 3.
In addition to the gridded distribution rate and the Poincaré plot, return
  • PIx [% points below the line of identity (LI)]
  • GIx [Proportion of point distances above the LI]
  • SIx [Ratio of phase angles of points above the LI, w.r.t. LI]
  • AIx [Ratio of the cumulative area of sectors of points above the LI]
X = Data(:,1);
[GDE, GDR, PIx, GIx, SIx, AIx] = GridEn(X, 'm', 9, 'tau', 3, 'Logx', 2, 'Plotx', true)
GDE = 4.1956
GDR = 0.3580
PIx = 49.3575
GIx = 49.8916
SIx = 49.9713
AIx = 49.8855
