Cosine Similarity Entropy: CoSiEn()

Table of Contents
Syntax: References: Note: Example 1: Example 2: Example 3: Example 4:

Syntax:

[CoSi, Bm] = CoSiEn(Sig, 'm', 2, 'tau', 1, 'r', 0.1, 'Logx', 2, 'Norm', 0)
  • Sig Time series signal, a vector of length > 10.
  • m Embedding dimension, an integer > 1.
  • tau Time delay, a positive integer.
  • r Angular threshold, a value in range [0 < r < 1]
  • Logx Logarithm base in Shannon's entropy formula, a positive scalar.
  • Norm Normalisation of Sig, an interger in range [0 4]: 0 - no normalisation (default) / 1 - median removed / 2 - mean removed / 3 - normalised by standard deviation / 4 - normalised to range [-1 1]
  • CoSi Cosime similarity entropy estimate.
  • Bm Sum of global probabilities.

References:

  1. Theerasak Chanwimalueang and Danilo Mandic, Cosine similarity entropy: Self-correlation-based complexity analysis of dynamical systems, Entropy 19.12 (2017): 652.

Note:

In the following examples, example signals / data are imported into MatLab using the ExampleData function from EntropyHub.
To use this function as outlined in the examples below, an internet connection is required.
ExampleData accepts any of the following strings:
  • 'uniform' uniformly distributed random numbers in range [0 1]
  • 'gaussian' normally distributed random numbers with mean = 0, and standard deviation = 1
  • 'randintegers' uniformly distributed pseudorandom integers in range [1 8]
  • 'chirp' chirp signal with the following parameters: f0 = .01, t1 = 4000, f1 = .025
  • 'lorenz' X, Y, Z components of the Lorenz system (sigma: 10, beta: 8/3, rho: 28) [Xo = 10, Yo = 20, Zo = 10]
  • 'henon' X, Y components of the Henon attractor (alpha: 1.4, beta: .3) [Xo = 0, Yo = 0]

Example 1:

Import a chirp signal and calculate the cosine similarity entropy with an embedding dimension (m) of 2.
X = ExampleData('chirp');
CoSi = CoSiEn(X)
CoSi =
0.962463840246602

Example 2:

Import a signal of pseudorandom integers in the range [1, 8] and calculate the cosine similarity entropy with an embedding dimension of 5, a time delay (tau) of 3, and an angular threshold (r) of 0.1 (default).
X = ExampleData('randintegers');
CoSi = CoSiEn(X, 'm', 5, 'tau', 3)
CoSi =
0.405682327404656
Calculate CoSi with an angular threshold of 0.66666...
CoSi = CoSiEn(X, 'm', 5, 'tau', 3, 'r', 1/3)
CoSi =
0.107489470942869

Example 3:

Calculate the cosine similarity entropy and the sum of global probabilities (Bm) of a sequence of normally distributed random numbers with an embedding dimension of 3, time delay of 3, angular threshold of 0.3 and a logarithm base of 3 (trits).
X = ExampleData('gaussian');
[CoSi, Bm] = CoSiEn(X, 'm', 3, 'tau', 3, 'r', 0.3, 'Logx', 3)
CoSi =
0.462944361090963
Bm =
0.205990990510463
Run the same calculation with X normalised by its standard deviation.
[CoSi, Bm] = CoSiEn(X, 'm', 3, 'tau', 3, 'r', 0.3, 'Logx', 3, 'Norm', 3)
CoSi =
0.462634277443655
Bm =
0.205738654861700
Run the same calcuclation with X normalised to the range [-1, 1].
[CoSi, Bm] = CoSiEn(X, 'm', 3, 'tau', 3, 'r', 0.3, 'Logx', 3, 'Norm', 4)
CoSi =
0.468592346276254
Bm =
0.210638426035136

Example 4:

Import the x, y, and z components of the Lorenz system of equations.
Data = ExampleData('lorenz');
figure('Color', 'k')
plot3(Data(:,1), Data(:,2), Data(:,3), 'g.')
xlabel('x-component','color','g'),
ylabel('y-component','color','g'),
zlabel('z-component','color','g'),
view(-10,10), set(gca,'color','k'), axis square
Calculate the cosine similarity entropy and global probabilities of the z-component in nats (natural logarithm) with an embedding dimension of 3, time delay of 4, and normalize z to have zero mean.
Z = Data(:,3);
[CoSi, Bm] = CoSiEn(Z, 'm', 3, 'tau', 4, 'Logx', 0, 'Norm', 2)
CoSi =
0.404944552967208
Bm =
0.139989571007722
