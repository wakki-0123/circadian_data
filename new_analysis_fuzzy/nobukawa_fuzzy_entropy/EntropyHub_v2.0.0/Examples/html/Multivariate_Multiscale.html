Multivariate Multiscale Entropies:

Multivariate Multiscale entropy functions estimate the multivariate multiscale entropy of a multivariate time series using any of the Multivariate entropy functions.
  • Multivariate Multiscale Entropy - MvMSEn() [including coarse, modified, and time-shift graining methods]
  • Composite Multivariate Multiscale Entropy - cMvMSEn() [including refined-composite multiscale entropy (rcMvMSEn)]

These multiscale entropy functions require two positional arguments:
  1. The time series matrix - N observations (rows) x M sequences (columns),
  2. A multiscale entropy object (Mobj) returned by MSobject().
MSobject() is used to specify the type of entropy and its parameters used in the multiscale entropy estimation. For example:
Data = [3.9, 4.1, -2.3; % Multivariate Time series matrix (M=3)
-1.8, 0.2, -9.6;
...
2.4, -1.7, 8.0];
Mobj = MSobject('MvFuzzEn', 'm', [4,3,4], 'tau', [1,2,3], 'Logx', 2); % A multiscale entropy object for Multivariate Fuzzy Entropy (MvFuzzEn) with several parameters.
MSx = MvMSEn(Data, Mobj, ...);
Any Base, Cross-, or Multivariate entropy function (and every function-specific parameter) is accepted by MSobject, but not bidimensional entropies.
