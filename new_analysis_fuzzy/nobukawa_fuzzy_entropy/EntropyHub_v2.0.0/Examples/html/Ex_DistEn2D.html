Syntax:

Dist2D = DistEn2D(Mat, 'm', floor(size(Mat)/10), 'tau', 1, 'Bins', 'sturges', 'Logx', 2, 'Norm', true, 'Lock', true)
  • Mat N x M matrix, where N, M > 10.
  • m Embedding dimension, [default: (floor(N/10) floor(M/10))] - an integer > 1 for square submatrix embedding, or a two-element vector of integers > 1 representing the height and width of the template submatrix.
  • tau Time delay, a positive integer.
  • Bins Histogram bin selection method, in integer > 1 indicating the number of bins or one of the following strings: "sturges", "sqrt", "rice", "doanes" [default: "sturges"]
  • Logx Logarithm base in Shannon's entropy formula, a positive scalar. (Enter 0 for natural logarithm)
  • Norm Normalization of Dist2D value: false - no normalisation / true - normalises w.r.t number of histogram bins (default)
  • Lock See note on matrix size locking true Matrix height (N) and width (M)
  • Dist2D Bidimensional distribution entropy estimate.

References:

  1. Luiz Eduardo Virgili Silva, et al., Two-dimensional sample entropy: Assessing image texture through irregularity, Biomedical Physics & Engineering Express 2.4 (2016): 045002.

About DistEn2D():

DistEn2D has an important keyword argument: Lock. Bidimensional entropy functions are "locked" by default (Lock == true) to only permit matrices with a maximum size of 128 x 128. The reason for this is because there are hundreds of millions of pairwise calculations performed in the estimation of bidimensional entropy, so memory errors often occur when storing data on RAM. e.g. For a matrix of size [200 x 200], an embedding dimension (m) = 3, and a time delay (tau) = 1, there are 753,049,836 pairwise matrix comparisons (6,777,448,524 elemental subtractions). To pass matrices with sizes greater than [128 x 128], set 'Lock' to false.
*** WARNING: unlocking the permitted matrix size may cause MatLab to crash.***

Note:

In the following examples, sample matrices are imported into MatLab using the ExampleData function from EntropyHub.
To use this function as outlined in the examples below, an internet connection is required.
ExampleData accepts any of the following strings:
  • 'uniform_Mat' uniformly distributed random numbers in range [0 1]
  • 'gaussian_Mat' normally distributed random numbers with mean = 0, and standard deviation = 1
  • 'randintegers_Mat' uniformly distributed pseudorandom integers in range [1 8]
  • 'mandelbrot_Mat' image of fractal generated from the mandelbrot set
  • 'entropyhub_Mat' image of the entropyhub logo

Example 1:

Import a matrix of normally distributed random numbers .
X = ExampleData('gaussian_Mat');
figure('Color','k'),
imshow(X,'InitialMagnification',500),
cmap = colormap('turbo'); cmap(:,1) = 0;
colormap(cmap)
Calculate the bidimensional distribution entropy using an template matrix (m) with dimensions , where N and M are the number of rows and columns of the input matrix (X), a distance threshold (r) value equal to twice the matrix standard deviation (), and using Sturges' bin selection method.
DE2D = DistEn2D(X)
Note: 1/26 bins were empty
DE2D = 0.6904
Change the size of the template matrix to [6 x 6].
DE2D = DistEn2D(X, 'm', 6)
Note: 1/26 bins were empty
DE2D = 0.6901

Example 2:

Import a matrix of uniformly distributed random numbers in the range [0, 1] and calculate the bidimensional distribution entropy with a template matrix of size [2 x 5], a time delay (tau) of 3 and a 30 histogram bins.
X = ExampleData('uniform_Mat');
figure('Color','k'),
imshow(X,[],'InitialMagnification',500),
cmap = colormap('turbo'); cmap(:,1) = 0;
colormap(cmap)
DE2D = DistEn2D(X, 'm', [2 5], 'tau', 3, 'Bins', 30)
DE2D = 0.8511

Example 3:

Import an image of a Mandelbrot fractal as a matrix.
X = ExampleData('mandelbrot_Mat');
figure('Color','k'),
imshow(X,[],'InitialMagnification',500),
colormap('hot')
Calculate the bidimensional distribution entropy in dits (logarithm base 10) with a template matrix of size [8 x 8], a time delay (tau) of 2 and using Rice's bin selection method.
DE2D = DistEn2D(X, 'm', 8, 'tau', 2, 'Bins', 'rice', 'Logx', 10)
Note: 115/629 bins were empty
DE2D = 0.8133

Example 4:

Import an image of the EntropyHub logo as a matrix and calculate the bidimensional distribution entropy with and without normalizing the entropy output w.r.t. the number of histogram bins, and using Doane's bin selection method.
X = ExampleData('entropyhub_Mat');
figure('Color','k'),
imshow(X,[],'InitialMagnification',500),
colormap('cool')
DE2D = DistEn2D(X, 'Bins', 'doanes')
DE2D = 0.2607
DE2D = DistEn2D(X, 'Bins', 'doanes', 'Norm', false)
DE2D = 1.3781
