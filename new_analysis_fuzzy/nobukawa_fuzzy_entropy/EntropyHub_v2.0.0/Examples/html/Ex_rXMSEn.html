Syntax:

[MSx, Ci] = rXMSEn(Sig1, Sig2, Mobj, 'Scales', 3, 'F_Order', 6, 'F_Num', 0.5, 'RadNew', 0, 'Plotx', false)
  • Sig1 Time series signal, a vector (N > 10).
  • Sig2 Time series signal, a vector (N > 10).
  • Mobj Multiscale entropy object
  • Scales Number of grained time scales, a positive integer.
  • F_Order Butterworth low-pass filter order, a positive integer > 1, (default: 6)
  • F_Num Numerator of Butterworth low-pass filter cutoff frequency, where [0 < F_Num < 1]. The cutoff frequency at each scale () becomes: (default: 0.5)
  • RadNew Radius rescaling method, an integer in the range [0 4]. When the Cross- entropy method specified by Mobj is XSampEn or XApEn, RadNew allows the radius threshold to be updated based on the grained signal at each time scale (). If a radius threshold value (r) is specified in Mobj, this becomes the rescaling coefficient, otherwise it is set to 0.2 (default). The value of RadNew specifies one of the following methods: 0 - no rescaling / 1 - Pooled Standard Deviation / 2 - Pooled Variance / 3 - Total Mean Absolute Deviation / 4 - Total Median Absolute Deviation
  • Plotx A plot of the multiscale entropy curve: true - Plots time scale vs entropy value / false - No plot.
  • MSx Multiscale cross-entropy estimate at each time scale, a vector of length Scales.
  • Ci Complexity index (area under the multiscale entropy curve), a scalar.

About rXMSEn():

To use the refined multiscale cross-entropy function (rXMSEn), one must first create a multiscale entropy object using MSobject() to specify the type of entropy and its parameters used in the multiscale entropy estimation.
rXMSEn() returns two results:
  1. MSx the cross-entropy estimate at each time scale, a vector of length Scales
  2. Ci the complexity index (area under the multiscale entropy curve), a scalar value

The 'RadNew' Argument:

Each multiscale entropy function accepts a 'RadNew' name/value pair argument to perform radius threshold rescaling at each grained time-scale if the Cross entropy method specified by Mobj is XSampEn or XApEn.
If a radius distance threshold value (r) is specified in Mobj, RadNew becomes the rescaling coefficient, otherwise it is set to 0.2 (default). The value of RadNew specifies one of the following methods:
0 No rescaling
1 Pooled Standard deviation -
2 Pooled Variance -
3 Total Mean Abs Deviation -
4 Total Median Abs Deviation -

Note:

In the following examples, example signals / data are imported into MatLab using the ExampleData function from EntropyHub.
To use this function as outlined in the examples below, an internet connection is required.
ExampleData accepts any of the following strings:
  • 'uniform2' uniformly distributed random numbers in range [0 1]
  • 'gaussian2' normally distributed random numbers with mean = 0, and standard deviation = 1
  • 'randintegers2' uniformly distributed pseudorandom integers in range [1 8]
  • 'lorenz' X, Y, Z components of the Lorenz system (sigma: 10, beta: 8/3, rho: 28) [Xo = 10, Yo = 20, Zo = 10]
  • 'henon' X, Y components of the Henon attractor (alpha: 1.4, beta: .3) [Xo = 0, Yo = 0]

Example 1:

Import two signals of uniformly distrobuted numbers in the range [0, 1] and create a multiscale entropy object with the following parameters:
EnType = XFuzzEn(), embedding dimension = 3, fuzzy function = 'triangular', function function parameter = 0.15
X = ExampleData('uniform2');
Mobj = MSobject('XFuzzEn', 'm', 3, 'Fx', 'triangular', 'r', 0.15)
Mobj = struct with fields:
Func: @XFuzzEn m: 3 r: 0.1500 Fx: 'triangular'
Calculate the refined multiscale cross-fuzzy entropy over 4 temporal scales.
MSx = rXMSEn(X(:,1), X(:,2), Mobj, 'Scales', 4)

Example 3:

Mobj = MSobject('XSampEn', 'm', 4, 'r', 1.25)
Mobj = struct with fields:
Func: @XSampEn m: 4 r: 1.2500
Calculate the refined multiscale cross-sample entropy and the complexity index over 5 temporal scales using a 3rd order Butterworth filter with a normalised corner frequency of at each temporal scale (), where the radius distancec threshold value (r) specified by Mobj becomes scaled by the median absolute deviation of the filtered signals at each scale.
[MSx, Ci] = rXMSEn(X(:,1), X(:,2), Mobj, 'Scales', 5, 'F_Order', 3, 'F_Num', 0.6, 'RadNew', 4)
