Syntax:

[Perm, Pnorm, cPE] = PermEn(Sig, 'm', 2, 'tau', 1, 'Typex', 'none', 'tpx', [], 'Logx', 2, 'Norm', false)
  • Sig Time series signal, a vector of length > 10. It is recommeneded that signal length, N > 5m! (Amigo et al., Europhys. Lett. 83:60005, 2008)
  • m Embedding dimension, an integer > 1.
  • tau Time delay, a positive integer.
  • Typex Variant of permutation entropy, one of the following strings: "finegrain" Fine-grained permutation entropy / "modified" Modified permutation entropy / "weighted" Weighted permutation entropy / "ampaware" Amplitude-aware permutation entropy / "edge" Edge permutation entropy / "uniquant" Uniform quantization-based permutation entropy / "phase" Phase permutation entropy
  • tpx Tuning parameter for the permutation entropy specified by the Typex argument. finegrain tpx is the α parameter, a positive scalar (default: 1) / ampaware tpx is the A parameter, a value in range [0 1] (default: 0.5) / edge tpx is the r sensitivity parameter, a scalar > 0 (default: 1) / uniquant tpx is the L parameter, an integer > 1 (default: 4) / phase tpx == 1 unwraps (shifts phase angle) the hilbert-transformed signal by pi radians, 0 otherwise.
  • Logx Logarithm base in Shannon's entropy formula, a positive scalar.
  • Norm Normalisation of Pnorm value, a boolean operator: false normalises w.r.t Log(# of permutation symbols [m]) - default / true normalises w.r.t Log(# of all possible permutations [m!]). * Note: Normalised permutation entropy is undefined for m = 1. ** Note: When Typex = 'uniquant' and Norm = true, normalisation of Perm is calculated w.r.t.
  • Perm Permutation entropy estimates for embedding dimensions 1:m.
  • Pnorm Normalised Permutation entropy estimates.
  • cPE Conditional permutation entropy.

About PermEn():

Permutation entropy is undefined for singular embeddings (m = 1). Thus, the first value in the Perm output will always be 0.

Note:

In the following examples, signals / data are imported into MatLab using the ExampleData function from EntropyHub.
To use this function as outlined in the examples below, an internet connection is required.
ExampleData accepts any of the following strings:
  • 'uniform' uniformly distributed random numbers in range [0 1]
  • 'gaussian' normally distributed random numbers with mean = 0, and standard deviation = 1
  • 'randintegers' uniformly distributed pseudorandom integers in range [1 8]
  • 'chirp' chirp signal with the following parameters: f0 = .01, t1 = 4000, f1 = .025
  • 'lorenz' X, Y, Z components of the Lorenz system (sigma: 10, beta: 8/3, rho: 28) [Xo = 10, Yo = 20, Zo = 10]
  • 'henon' X, Y components of the Henon attractor (alpha: 1.4, beta: .3) [Xo = 0, Yo = 0]

Example 1:

Import a signal of normally distributed random numbers , and calculate the permutation entropy for embedding dimensions (m) equal to 1 and 2, with all other default parameters.
X = ExampleData('gaussian');
Perm = PermEn(X)
Perm = 1×2
0 1.0000

Example 2:

Import a chirp signal and calculate the permutation entropy for each embedding dimension (m) from 1 to 4.
X = ExampleData('chirp');
Perm = PermEn(X, 'm', 4)
Perm = 1×4
0 1.0000 1.2739 1.5457
Select the last value of Perm to get the permutation entropy for m = 4.
Perm(end)
ans = 1.5457
Calculate the permutation entropy and normalised permutation entropy (Pnorm) of the same signal for a maximum embedding dimension of 5 and a time delay (tau) of 4 samples.
[Perm, Pnorm] = PermEn(X, 'm', 5, 'tau', 4)
Perm = 1×5
0 1.0000 1.7760 2.5167 3.2094
Pnorm = 1×5
NaN 1.0000 0.8880 0.8389 0.8024

Example 3:

Calculate the normalised permutation entropy of the same chirp signal normalized w.r.t the number of all permutations (with - m!) and w.r.t. the number of symbols (without - m), using the uniform-quantization method (default) and a maximum embedding dimension of 3.
[~, Pnorm_with] = PermEn(X, 'm', 3, 'Norm', true)
Pnorm_with = 1×3
NaN 1.0000 0.4928
[~, Pnorm_without] = PermEn(X, 'm', 3, 'Norm', false)
Pnorm_without = 1×3
NaN 1.0000 0.6370
[**Note that Pnorm is undefined for m = 1]

Example 4:

Import a signal of pseudorandom integers in the range [1, 8].
X = ExampleData('randintegers');
Calculate the amplitude-aware permutation entropy and the normalised amplitude-aware permutation entropy (Pnorm) with a maximum embedding dimension of 4 and time delay (tau) of 3.
[Perm, Pnorm] = PermEn(X, 'm', 4, 'tau', 3, 'Typex', 'ampaware')
Perm = 1×4
0 0.9950 2.5562 4.5069
Pnorm = 1×4
NaN 0.9950 1.2781 1.5023
Change the A parameter (tpx) of amplitude-aware permuation entropy to 0.8:
[Perm, Pnorm] = PermEn(X, 'm', 4, 'tau', 3, 'Typex', 'ampaware','tpx', 0.8)
Perm = 1×4
0 0.9905 2.5511 4.5010
Pnorm = 1×4
NaN 0.9905 1.2755 1.5003

Example 5:

Import a signal of uniformly distributed random numbers in the range [0, 1] and calculate the weighted permutation entropy for a maximum embedding dimension of 2.
X = ExampleData('uniform');
Perm = PermEn(X, 'Typex', 'weighted')
Perm = 1×2
0 1.0000
Calculate edge permutation entropy with an r sensitivity parameter = pi, with embedding dimension of 6 and time delay of 4.
Perm = PermEn(X, 'm', 6, 'tau', 4, 'Typex', 'edge', 'tpx', pi)
Perm = 1×6
0 1.0000 2.5806 4.5778 6.8844 9.3752

Example 6:

Import the x and y components of the Henon system of equations.
Data = ExampleData('henon');
figure('Color', 'k')
plot(Data(:,1), Data(:,2), 'g.')
xlabel('x-component','color','g'), ylabel('y-component','color','g')
set(gca,'color','k'), axis square
Calculate the modified permutation entropy of the y component in dits (logarithm base 10) with an embedding dimension of 4 and return Pnorm normalised w.r.t the number of all possible permutations (m!).
Y = Data(:,2);
[Perm, Pnorm] = PermEn(Y, 'm', 4, 'Typex', 'modified', 'Logx', 10, 'Norm', false)
Perm = 1×4
0 0.2949 0.6873 0.9497
Pnorm = 1×4
NaN 0.2949 0.3436 0.3166

Example 7:

Import the x, y, and z components of the Lorenz system of equations.
Data = ExampleData('lorenz');
figure('Color', 'k')
plot3(Data(:,1), Data(:,2), Data(:,3), 'g.')
xlabel('x-component','color','g'),
ylabel('y-component','color','g'),
zlabel('z-component','color','g'),

Calculate fine-grained permutation entropy of the z component in dits (logarithm base 10) with an embedding dimension of 3, time delay of 2, an alpha parameter of 1.234, returning Pnorm normalised w.r.t the number of all possible permutations (m!) and the condition permutation entropy (cPE) estimate.
Z = Data(:,3);
[Perm, Pnorm, cPE] = PermEn(Z, 'm', 3, 'tau', 2, 'Typex', 'finegrain', 'tpx', 1.234, 'Logx', 10, 'Norm', false)

0 0.8687 0.9468
