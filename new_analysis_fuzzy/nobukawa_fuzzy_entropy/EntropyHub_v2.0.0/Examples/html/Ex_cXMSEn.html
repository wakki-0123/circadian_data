Syntax:

[MSx, Ci] = cXMSEn(Sig1, Sig2, Mobj, 'Scales', 3, 'RadNew', 0, 'Refined', false, 'Plotx', false)
  • Sig1 Time series signal, a vector (N > 10).
  • Sig2 Time series signal, a vector (N > 10).
  • Mobj Multiscale entropy object
  • Scales Number of grained time scales, a positive integer.
  • RadNew Radius rescaling method, an integer in the range [0 4]. When the Cross- entropy method specified by Mobj is XSampEn or XApEn, RadNew allows the radius threshold to be updated based on the grained signal at each time scale (). If a radius threshold value (r) is specified in Mobj, this becomes the rescaling coefficient, otherwise it is set to 0.2 (default). The value of RadNew specifies one of the following methods: 0 - no rescaling / 1 - Pooled Standard Deviation / 2 - Pooled Variance / 3 - Total Mean Absolute Deviation / 4 - Toal Median Absolute Deviation
  • Refined When Refined == true and the entropy function (EnType) contained in Mobj is XSampEn, cXMSEn returns the refined-composite multiscale cross-sample entropy (rcXMSEn). When Refined == true and the entropy function (EnType) contained in Mobj is XFuzzEn, cXMSEn returns the refined-composite multiscale cross-fuzzy entropy (link).
  • Plotx A plot of the multiscale entropy curve: true - Plots time scale vs entropy value / false - No plot.
  • MSx Multiscale entropy estimate at each time scale, a vector of length Scales.
  • Ci Complexity index (area under the multiscale entropy curve), a scalar.

About cXMSEn():

To use the composite multiscale cross-entropy function (cXMSEn), one must first create a multiscale entropy object using MSobject() to specify the type of entropy and its parameters used in the multiscale cross-entropy estimation.
cXMSEn() returns two results:
  1. MSx the entropy estimate at each time scale, a vector of length Scales
  2. Ci the complexity index (area under the multiscale entropy curve), a scalar value

The 'Refined' Argument:

When the type of cross-entropy specified by the multiscale entropy object (Mobj) is 'XSampEn', the refined-composite multiscale cross-sample entropy can be calculated by setting the 'Refined' name/value argument to true (see Example 4). Alternatively, when the type of cross-entropy specified by the multiscale entropy object (Mobj) is 'XFuzzEn', the refined-composite multiscale cross-fuzzy entropy is calculated if 'Refined' is true.
Refined-composite multiscale cross-entropy sums the number of match state vectors for each grained time series before prior to calculating the Shannon entropy.

The 'RadNew' Argument:

Each multiscale cross-entropy function accepts a 'RadNew' name/value pair argument to perform radius distance threshold rescaling at each grained time-scale if the Cross entropy method specified by Mobj is XSampEn or XApEn.
If a radius threshold value (r) is specified in Mobj, RadNew becomes the rescaling coefficient, otherwise it is set to 0.2 (default). The value of RadNew specifies one of the following methods:
0 No rescaling
1 Pooled Standard deviation -
2 Pooled Variance -
3 Total Mean Abs Deviation -
4 Total Median Abs Deviation -

Note:

In the following examples, signals / data are imported into MatLab using the ExampleData function from EntropyHub.
To use this function as outlined in the examples below, an internet connection is required.
ExampleData accepts any of the following strings:
  • 'uniform2' uniformly distributed random numbers in range [0 1]
  • 'gaussian2' normally distributed random numbers with mean = 0, and standard deviation = 1
  • 'randintegers2' uniformly distributed pseudorandom integers in range [1 8]
  • 'lorenz' X, Y, Z components of the Lorenz system (sigma: 10, beta: 8/3, rho: 28) [Xo = 10, Yo = 20, Zo = 10]
  • 'henon' X, Y components of the Henon attractor (alpha: 1.4, beta: .3) [Xo = 0, Yo = 0]

Example 1:

Import two signals of normally distributed random numbers [] and create a multiscale entropy object with the following parameters:
EnType = XPermEn(), embedding dimension = 3, time delay = 2
X = ExampleData('gaussian2');
Mobj = MSobject('XPermEn', 'm', 3, 'tau', 2)
Mobj = struct with fields:
Func: @XPermEn m: 3 tau: 2
Calculate the composite multiscale cross-permutation entropy over 3 temporal scales.
MSx = cXMSEn(X(:,1), X(:,2), Mobj)
. . . . . . .
MSx = 1×3
0.6409 0.6397 0.6423
Calculate the composite multiscale cross-permutation entropy over 5 scales instead.
MSx = cXMSEn(X(:,1), X(:,2), Mobj, 'Scales', 5)
. . . . . . . . . . . . . . . .
MSx = 1×5
0.6409 0.6397 0.6423 0.6313 0.6263

Example 2:

Import two signals of unifromly distributed pseudorandom integers in the range [1 8] and create a multiscale entropy object with the following parameters:
EnType = XApEn(), embedding dimension = 2, time delay = 2, radius distance threshold = 0.5.
X = ExampleData('randintegers2');
Mobj = MSobject('XApEn', 'm', 2, 'tau', 2, 'r', 0.5)
Mobj = struct with fields:
Func: @XApEn m: 2 tau: 2 r: 0.5000
Calculate the comsposite multiscale cross-approximate entropy over 3 temporal scales where the radius distance threshold value (r) specified by Mobj becomes scaled by the pooled variance of the signal at each scale.
MSx = cXMSEn(X(:,1), X(:,2), Mobj, 'Scales', 3, 'RadNew', 2)
. . . . . . .
MSx = 1×3
0.6583 0.9953 1.1677

Example 3:

Import two signals of uniformly distributed random numbers in the range [0, 1] and create a multiscale entropy object with the following parameters:
EnType = XDistEn(), embedding dimension = 6, time delay = 5, number of histogram bins = 4, logarithm base = 3, normalisation = true
X = ExampleData('uniform2');
Mobj = MSobject('XDistEn', 'm', 6, 'tau', 5, 'Bins', 4, 'Logx', 3, 'Norm', true)
Mobj = struct with fields:
Func: @XDistEn m: 6 tau: 5 Bins: 4 Logx: 3 Norm: 1
Calculate the composite multiscale cross-distribution entropy and the corresponding complexity index over 3 temporal scales, also returning a plot or the multiscale entropy curve.
[MSx, Ci] = cXMSEn(X(:,1), X(:,2), Mobj, 'Plotx', true)
. . . . . . .
MSx = 1×3
0.7712 0.7035 0.6550
Ci = 2.1297

Example 4:

Import the x, y, and z components of the Lorenz system of equations and create a multiscale entropy object with the following parameters:
EnType = XSampEn(), embedding dimension = 2, time delay = 1, radius threshold = 0.2*SD(YZ)
Data = ExampleData('lorenz');
figure('Color', 'k')
plot3(Data(:,1), Data(:,2), Data(:,3), 'g.')
xlabel('x-component','color','g'),
ylabel('y-component','color','g'),
zlabel('z-component','color','g'),
view(-10,10), set(gca,'color','k'), axis square
% The specified parameters are the default parameters for XSampEn
Mobj = MSobject('XSampEn')
Mobj = struct with fields:
Func: @XSampEn
Calculate the refined-composite multiscale cross-sample entropy and complexity index betweem the y and z-components over 5 temporal scales and return a plot of the multscale entropy curve.
Y = Data(:,2);
Z = Data(:,3);
 
[MSx, Ci] = cXMSEn(Y, Z, Mobj, 'Scales', 5, 'Refined', true, 'Plotx', true)
. . . . . . . . . . . . . . . .
MSx = 1×5
0.7014 1.1274 3.0063 5.1039 6.0552
Ci = 15.9941
