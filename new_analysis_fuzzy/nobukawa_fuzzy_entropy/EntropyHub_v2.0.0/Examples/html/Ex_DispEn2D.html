Bidimensional Dispersion Entropy: DispEn2D()

Syntax:

[Disp2D, RDE] = DispEn2D(Mat, 'm', floor(size(Mat)/10), 'tau', 1, 'c', 3, 'Typex', 'ncdf', 'Logx', exp(1), 'Norm', false, 'Lock', true)
  • Mat N x M matrix, where N, M > 10.
  • m Embedding dimension, [default: (floor(N/10) floor(M/10))] - an integer > 1 for square submatrix embedding, or a two-element vector of integers > 1 representing the height and width of the template submatrix.
  • tau Time delay, a positive integer.
  • c Number of symbols in transform, an integer > 1.
  • Typex Type of symbolic sequence transform, one of the following strings: "ncdf" Normalised cumulative distribution function / "kmeans" K-means clustering algorithm (**Note: The "kmeans" algorithm uses random initialization conditions. This causes results to vary slightly each time it is called.) / "linear" Linear segmentation of signal range / "equal" Approx. equal number of symbols.
  • Logx Logarithm base in Shannon's entropy formula, a positive scalar.
  • Norm Normalization of Disp2D and RDE values: false - no normalisation / true - normalises w.r.t number of of unique dispersion patterns (default)
  • Lock See note on matrix size locking true Matrix height (N) and width (M)
  • Disp2D Bidimensional dispersion entropy estimate.
  • RDE Reverse bidimensional dispersion entropy estimate.

References:

  1. Hamed Azami, et al., Two-dimensional dispersion entropy: An information-theoretic method for irregularity analysis of images, Signal Processing: Image Communication, (2019) 75: 178-187

About DispEn2D():

DispEn2D has an important keyword argument: Lock. Bidimensional entropy functions are "locked" by default (Lock == true) to only permit matrices with a maximum size of 128 x 128. To pass matrices with sizes greater than [128 x 128], set 'Lock' to false.
*** WARNING: unlocking the permitted matrix size may cause MatLab to crash.***

Note:

In the following examples, sample matrices are imported into MatLab using the ExampleData function from EntropyHub.
To use this function as outlined in the examples below, an internet connection is required.
ExampleData accepts any of the following strings:
  • 'uniform_Mat' uniformly distributed random numbers in range [0 1]
  • 'gaussian_Mat' normally distributed random numbers with mean = 0, and standard deviation = 1
  • 'randintegers_Mat' uniformly distributed pseudorandom integers in range [1 8]
  • 'mandelbrot_Mat' image of fractal generated from the mandelbrot set
  • 'entropyhub_Mat' image of the entropyhub logo

Example 1:

Import a matrix of uniformly distributed random numbers in the range [0, 1].
X = ExampleData('uniform_Mat');
figure('Color','k'),
imshow(X,'InitialMagnification',500),
cmap = colormap('turbo'); cmap(:,1) = 0;
colormap(cmap)
Calculate the bidimensional dispersion entropy using an template matrix (m) with dimensions , where N and M are the number of rows and columns of the input matrix (X), a 3-symbol data transform using the normal cumulative distribution function (NCDF).
Disp2D = DispEn2D(X)
Disp2D =
7.657282792978227
Change the size of the template matrix to [4 x 4].
Disp2D = DispEn2D(X, 'm', 4)
Disp2D =
7.700295203420202

Example 2:

Import a matrix of uniformly distributed random integers in the range [1, 8] and calculate the bidimensional dispersion entropy and reverse bidimensional dispersion entropy (RDE) with a template matrix of size [4 x 3], a time delay (tau) of 3, a 4-symbol data transform with an approximately equal number of symbols.
X = ExampleData('randintegers_Mat');
figure('Color','k'),
imshow(X,[],'InitialMagnification',500),
cmap = colormap('turbo'); cmap(:,1) = 0;
colormap(cmap)
[Disp2D, RDE] = DispEn2D(X, 'm', [4, 3], 'tau', 3, 'c', 4, 'Typex', 'equal')
Disp2D =
8.775953099366534
RDE =
1.542971053645682e-04

Example 3:

Import an image of a Mandelbrot fractal as a matrix.
X = ExampleData('mandelbrot_Mat');
figure('Color','k'),
imshow(X,[],'InitialMagnification',500),
colormap('hot')
Calculate the bidimensional dispersion entropy in trits (logarithm base 3) with a template matrix of size [2 x 2], a time delay (tau) of 2, with a 7-symbol data transform using K-means clustering and normalise the entropy value with respect to the number of unique dispersion patterns found.
Disp2D = DispEn2D(X, 'm', 2, 'tau', 2, 'c', 7, 'Typex', 'kmeans', 'Logx', 3, 'Norm', true)
Disp2D =
0.898270682368869
