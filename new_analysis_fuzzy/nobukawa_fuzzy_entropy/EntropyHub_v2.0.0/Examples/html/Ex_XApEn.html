Cross-Approximate Entropy: XApEn()

Table of Contents
Syntax: References: About XApEn(): Note: Example 1: Example 2: Example 3: Example 4:

Syntax:

[XAp, Phi] = XApEn(Sig1, Sig2, 'm', 2, 'tau', 1, 'r', 0.2*std_pooled(Sig1, Sig2), 'Logx', exp(1))
  • Sig1 Time series signal, a vector (N > 10).
  • Sig2 Time series signal, a vector (N > 10).
  • m Embedding dimension, a positive integer.
  • tau Time delay, a positive integer.
  • r Distance threshold, a positive scalar.
  • Logx Logarithm base in Shannon's entropy formula, a positive scalar.
  • XAp Cross-approximate entropy estimates, a vector of length m+1. **The first value of XAp is the zeroth estimate, i.e. , and the last value of XAp is the estimate for the specified m.
  • Phi The number of matched state vectors for each embedding dimension from 0 to m+1.

References:

  1. Steven M. Pincus, Approximate entropy as a measure of system complexity, Proceedings of the National Academy of Sciences, 88.6 (1991): 2297-2301

About XApEn():

XApEn is direction-dependent. Thus, the first column of Sig is used as the template data sequence, and the second column is the matching sequence.
The first value of XAp is the zeroth estimate, i.e. , and the last value of XAp is the estimate for the specified m.

Note:

In the following examples, signals / data are imported into MatLab using the ExampleData function from EntropyHub.
To use this function as outlined in the examples below, an internet connection is required.
ExampleData accepts any of the following strings:
  • 'uniform2' uniformly distributed random numbers in range [0 1]
  • 'gaussian2' normally distributed random numbers with mean = 0, and standard deviation = 1
  • 'randintegers2' uniformly distributed pseudorandom integers in range [1 8]
  • 'lorenz' X, Y, Z components of the Lorenz system (sigma: 10, beta: 8/3, rho: 28) [Xo = 10, Yo = 20, Zo = 10]
  • 'henon' X, Y components of the Henon attractor (alpha: 1.4, beta: .3) [Xo = 0, Yo = 0]

Example 1:

Import two signals of uniformly distributed random numbers in the range [0,1] and calculate the cross-approximate entropy for embedding dimensions (m) 0, 1 and 2.
X = ExampleData('uniform2');
XAp = XApEn(X(:,1), X(:,2))
XAp = 13
2.200661178812759 2.208000809123448 2.209394608853056

Example 4:

Import the x, y and z components of the Lorenz system of equations.
Data = ExampleData('lorenz');
figure('Color', 'k')
plot3(Data(:,1), Data(:,2), Data(:,3), 'g.')
xlabel('x-component','color','g'),
ylabel('y-component','color','g'),
zlabel('z-component','color','g'),
view(-10,10), set(gca,'color','k'), axis square
Calculate the cross-approximate entropy between the x and y components in bits (logarithm base 2) and return the log-normalised number of matched state vectors (Phi) with all other default parameter values.
X = Data(:, 1);
Y = Data(:, 2);
[XAp, Phi] = XApEn(X, Y, 'Logx', 2)
