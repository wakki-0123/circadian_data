Syntax:

[XSamp, A, B, (Vcp, Ka, Kb)] = XSampEn(Sig1, Sig2, 'm', 2, 'tau', 1, 'r', 0.2*std_pooled(Sig1, Sig2), 'Logx', exp(1), 'Vcp', false)
  • Sig1 Time series signal, a vector (N > 10).
  • Sig2 Time series signal, a vector (N > 10).
  • m Embedding dimension, a positive integer.
  • tau Time delay, a positive integer.
  • r Distance threshold , a positive scalar.
  • Logx Logarithm base in Shannon's entropy formula, a positive scalar.
  • Vcp Option to return variance of conditional probabilities and the number of overlapping matching vector pairs.
  • XSamp Sample entropy estimates, a vector of length m+1. **The first value of XSamp is the zeroth estimate, i.e. , and the last value of XSamp is the estimate for the specified m.
  • A The number of matched state vectors for each embedding dimension from 0 to m.
  • B The number of matched state vectors for each embedding dimension from 1 to m+1.
  • [Vcp, Ka, Kb] The conditional (Vcp), i.e. CP = A/B, and the number of overlapping matching vector pairs of lengths m+1 (Ka) and m (Kb), respectively.

References:

  1. Joshua S Richman and J. Randall Moorman, Physiological time-series analysis using approximate entropy and sample entropy, American Journal of Physiology-Heart and Circulatory Physiology (2000).
  2. Douglas E Lake, Joshua S Richman, M.P. Griffin, J. Randall Moorman, Sample entropy analysis of neonatal heart rate variability, American Journal of Physiology-Regulatory, Integrative and Comparative Physiology, 283, no. 3 (2002): R789-R797.

About XSampEn():

The first value of XSamp is the zeroth estimate, i.e. , and the last value of XSamp is the estimate for the specified m.

Note:

In the following examples, signals / data are imported into MatLab using the ExampleData function from EntropyHub.
To use this function as outlined in the examples below, an internet connection is required.
ExampleData accepts any of the following strings:
  • 'uniform2' uniformly distributed random numbers in range [0 1]
  • 'gaussian2' normally distributed random numbers with mean = 0, and standard deviation = 1
  • 'randintegers2' uniformly distributed pseudorandom integers in range [1 8]
  • 'lorenz' X, Y, Z components of the Lorenz system (sigma: 10, beta: 8/3, rho: 28) [Xo = 10, Yo = 20, Zo = 10]
  • 'henon' X, Y components of the Henon attractor (alpha: 1.4, beta: .3) [Xo = 0, Yo = 0]
