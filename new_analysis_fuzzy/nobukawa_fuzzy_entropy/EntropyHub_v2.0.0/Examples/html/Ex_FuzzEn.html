Fuzzy Entropy: FuzzEn()

Syntax:

[Fuzz, Ps1, Ps2] = FuzzEn(Sig, 'm', 2, 'tau', 1, 'Fx', 'default', 'r', [0.2, 2], 'Logx', exp(1))
  • Sig Time series signal, a vector of length > 10.
  • m Embedding dimension, a positive integer.
  • tau Time delay, a positive integer.
  • Fx Type of fuzzy function for distance transformation, one of the following strings: 'default' / 'sigmoid' / 'modsampen' / 'gudermannian' / 'triangular' / 'trapezoidal1' / 'trapezoidal2' / 'z_shaped' / 'bell' / 'gaussian' / 'constgaussian'. See the EntropyHub Guide (.pdf) for more info.
  • r Parameters of the fuzzy function specified by Fx, a 1 element scalar or a 2 element tuple of positive values depending on the fuzzy function as shown above. Default/Sigmoid/ModSampEn/Trapezoidal2/Bell: Two element vector; Gudermannian/Triangular/Trapezoidal1/Gaussian/Constgaussian/Z-Shaped: A scalar value
  • Logx Logarithm base in Shannon's entropy formula, a positive scalar.
  • Fuzz Fuzzy entropy estimates for each embedding dimension 1:m.
  • Ps1 The average fuzzy distances for embedding dimensions 1:m.
  • Ps2 The average fuzzy distances for embedding dimensions 2:m+1.

References:

  1. Weiting Chen, et al., Characterization of surface EMG signal based on fuzzy entropy, IEEE Transactions on neural systems and rehabilitation engineering, 15.2 (2007): 266-272
  2. Hong-Bo Xie, Wei-Xing He, and Hui Liu, Measuring time series regularity using nonlinear similarity-based sample entropy, Physics Letters A, 372.48 (2008): 7140-7146
  3. Hamed Azami, et al. Fuzzy Entropy Metrics for the Analysis of Biomedical Signals: Assessment and Comparison, IEEE Access, 7 (2019): 104833-104847

Note:

In the following examples, example signals / data are imported into MatLab using the ExampleData function from EntropyHub.
To use this function as outlined in the examples below, an internet connection is required.
ExampleData accepts any of the following strings:
  • 'uniform' uniformly distributed random numbers in range [0 1]
  • 'gaussian' normally distributed random numbers with mean = 0, and standard deviation = 1
  • 'randintegers' uniformly distributed pseudorandom integers in range [1 8]
  • 'chirp' chirp signal with the following parameters: f0 = .01, t1 = 4000, f1 = .025
  • 'lorenz' X, Y, Z components of the Lorenz system (sigma: 10, beta: 8/3, rho: 28) [Xo = 10, Yo = 20, Zo = 10]
  • 'henon' X, Y components of the Henon attractor (alpha: 1.4, beta: .3) [Xo = 0, Yo = 0]

Example 1:

Import a signal of uniformly distributed random numbers in the range [0, 1] and calculate the fuzzy entropy for embedding dimensions (m) equal to 1 and 2.
X = ExampleData('uniform');
Fuzz = FuzzEn(X)
Fuzz = 12
0.307996349620718 0.408481478749948

Example 2:

Import a signal of normally distributed random numbers , and calculate the fuzzy entropy for each embedding dimension (m) from 1 to 4.
X = ExampleData('gaussian');
Fuzz = FuzzEn(X, 'm', 4)
Fuzz = 14
1.187779936020232 1.369169066016786 1.121458693076394 1.028011286657045
Select the last value to get the fuzzy entropy for m = 4.
Fuzz(end)
ans =
1.028011286657045
Calculate the fuzzy entropy of the same signal for embedding dimensions (m) 1 to 4 with a time delay (tau) of 2 samples.
Fuzz = FuzzEn(X, 'm', 4, 'tau', 2)
Fuzz = 14
1.198126893589655 1.369375690085505 1.114705362234607 1.027077602989844

Example 4:

Import a chirp signal and calculate the fuzzy entropy using a 'bell' fuzzy function with the r parameters ( = 0.1, =0.2):
X = ExampleData('chirp');
Fuzz = FuzzEn(X, 'Fx', 'bell', 'r', [.1, .2])
Fuzz = 12
0.513377770062839 0.126569334774036
Change the fuzzy function to 'trapezoidal2' and use the same 'r' parameter values:
Fuzz = FuzzEn(X, 'Fx', 'trapezoidal2', 'r', [.1, .2])

Example 5:
