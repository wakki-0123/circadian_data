Syntax:

[MSx, Ci] = rMSEn(Sig, Mobj, 'Scales', 3, 'F_Order', 6, 'F_Num', 0.5, 'RadNew', 0, 'Plotx', false)
  • Sig Time series signals, a vector of length > 10.
  • Mobj Multiscale entropy object
  • Scales Number of grained time scales, a positive integer.
  • F_Order Butterworth low-pass filter order, a positive integer > 1, (default: 6)
  • F_Num Numerator of Butterworth low-pass filter cutoff frequency, where [0 < F_Num < 1]. The cutoff frequency at each scale (τ) becomes: (default: 0.5)
  • RadNew Radius rescaling method, an integer in the range [0 4]. When the Base entropy method specified by Mobj is SampEn or ApEn, RadNew allows the radius threshold to be updated based on the grained signal at each time scale (). If a radius threshold value (r) is specified in Mobj, this becomes the rescaling coefficient, otherwise it is set to 0.2 (default). The value of RadNew specifies one of the following methods: 0 - no rescaling / 1 - Standard Deviation / 2 - Variance / 3 - Mean Absolute Deviation / 4 - Median Absolute Deviation
  • Plotx A plot of the multiscale entropy curve: true - Plots time scale vs entropy value / false - No plot.
  • MSx Multiscale entropy estimate at each time scale, a vector of length Scales.
  • Ci Complexity index (area under the multiscale entropy curve), a scalar.

References:

  1. José Fernando Valencia, et al., Refined multiscale entropy: Application to 24-h holter recordings of heart period variability in healthy and aortic stenosis subjects, IEEE Transactions on Biomedical Engineering 56.9 (2009): 2202-2213.
  2. Puneeta Marwaha and Ramesh Kumar Sunkaria, Optimal selection of threshold value 'r' for refined multiscale entropy, Cardiovascular engineering and technology 6.4 (2015): 557-576.

About rMSEn():

To use the refined multiscale entropy function (rMSEn), one must first create a multiscale entropy object using MSobject() to specify the type of entropy and its parameters used in the multiscale entropy estimation.
rMSEn() returns two results:
  1. MSx the entropy estimate at each time scale, a vector of length Scales
  2. Ci the complexity index (area under the multiscale entropy curve), a scalar value

The 'RadNew' Argument:

Each multiscale entropy function accepts a 'RadNew' name/value pair argument to perform radius threshold rescaling at each grained time-scale if the Base entropy method specified by Mobj is SampEn or ApEn.
If a radius threshold value (r) is specified in Mobj, RadNew becomes the rescaling coefficient, otherwise it is set to 0.2 (default). The value of RadNew specifies one of the following methods:
0 No rescaling
1 Standard deviation -
2 Variance -
3 Mean Abs Deviation -
4 Median Abs Deviation -

Note:

In the following examples, signals / data are imported into MatLab using the ExampleData function from EntropyHub.
To use this function as outlined in the examples below, an internet connection is required.
ExampleData accepts any of the following strings:
  • 'uniform' uniformly distributed random numbers in range [0 1]
  • 'gaussian' normally distributed random numbers with mean = 0, and standard deviation = 1
  • 'randintegers' uniformly distributed pseudorandom integers in range [1 8]
  • 'lorenz' X, Y, Z components of the Lorenz system (sigma: 10, beta: 8/3, rho: 28) [Xo = 10, Yo = 20, Zo = 10]
  • 'henon' X, Y components of the Henon attractor (alpha: 1.4, beta: .3) [Xo = 0, Yo = 0]

Example 1:

Import a chirp signal and create a multiscale entropy object with the following parameters:
EnType = SyDyEn(), embedding dimension = 3, time delay = 4, number of symbols = 13, symbolic sequence transform = 'uniform'
X = ExampleData('chirp');
Mobj = MSobject('SyDyEn', 'm', 3, 'tau', 4, 'c', 13, 'Typex', 'uniform')
Mobj = struct with fields:
Func: @SyDyEn m: 3 tau: 4 c: 13 Typex: 'uniform'
Calculate the refined multiscale symbolic dynamic entropy over 3 temporal scales.
MSx = rMSEn(X, Mobj, 'Scales', 3)

1.8076 2.0285 2.0099

Example 3:

Import a signal of uniformly distributed pseudorandom integers in the range [1, 8] and create a multiscale entropy object with the following parameters:
EnType = SampEn(), embedding dimension = 4, radius threshold = 1.25
X = ExampleData('randintegers');
Mobj = MSobject('SampEn', 'm', 4, 'r', 1.25)
Mobj = struct with fields:
Func: @SampEn m: 4 r: 1.2500
Calculate the refined multiscale sample entropy and the complexity index over 5 temporal scales using a 3rd order Butterworth filter with a normalised corner frequency of at each temporal scale (τ), where the radius threshold value (r) specified by Mobj becomes scaled by the median absolute deviation of the filtered signal at each scale.
[MSx, Ci] = rMSEn(X, Mobj, 'Scales', 5, 'F_Order', 3, 'F_Num', 0.6, 'RadNew', 4)
