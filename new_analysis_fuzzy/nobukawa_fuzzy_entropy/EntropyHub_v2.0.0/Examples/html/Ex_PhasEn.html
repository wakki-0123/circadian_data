Phase Entropy: PhasEn()

Table of Contents
Syntax: References: Note: Example 1: Example 2: Example 3:

Syntax:

Phas = PhasEn(Sig, 'K', 4, 'tau', 1, 'Logx', exp(1), 'Norm', true, 'Plotx', false)
  • Sig Time series signal, a vector of length > 10.
  • K Number of angular partitions, an integer > 1. **Note: Angular partitions of the second-order difference plot (SODP) are first split between 0 and n degrees w.r.t. the positive x-axis. Thus, even-numbered partitions are recommended for symmetry.
  • tau Time delay, a positive integer.
  • Logx Logarithm base in Shannon's entropy formula, a positive scalar.
  • Norm Normalisation of Phas: false - no normalisation (default) / true - normalises w.r.t. the number of partitions Log(K)
  • Plotx When Plotx == true, returns SODP (default: false)
  • Phas Phase entropy estimate.

References:

  1. Ashish Rohila and Ambalika Sharma, Phase entropy: a new complexity measure for heart rate variability, Physiological measurement 40.10 (2019): 105006.

Note:

In the following examples, signals / data are imported into MatLab using the ExampleData function from EntropyHub.
To use this function as outlined in the examples below, an internet connection is required.
ExampleData accepts any of the following strings:
  • 'uniform' uniformly distributed random numbers in range [0 1]
  • 'gaussian' normally distributed random numbers with mean = 0, and standard deviation = 1
  • 'randintegers' uniformly distributed pseudorandom integers in range [1 8]
  • 'chirp' chirp signal with the following parameters: f0 = .01, t1 = 4000, f1 = .025
  • 'lorenz' X, Y, Z components of the Lorenz system (sigma: 10, beta: 8/3, rho: 28) [Xo = 10, Yo = 20, Zo = 10]
  • 'henon' X, Y components of the Henon attractor (alpha: 1.4, beta: .3) [Xo = 0, Yo = 0]

Example 1:

Import a signal of normally distributed random numbers , and calculate the phase entropy with 4 angular partitions..
X = ExampleData('gaussian');
Phas = PhasEn(X)
Phas = 0.8033
Plot the secod-order difference plot with the output.
Phas = PhasEn(X, 'Plotx', true)
Phas = 0.8033

Example 2:

Import a chirp signal and calculate the phase entropy with 7 angular partitions (K), a time delay (tau) of 3, and normalize the estimate w.r.t. the log-number of partitions.
X = ExampleData('chirp');
Phas = PhasEn(X, 'K', 7, 'tau', 3)
Phas = 0.7118
Calculate the phase entropy with the same parameters without normalization.
Phas = PhasEn(X, 'K', 7, 'tau', 3, 'Norm', false)
Phas = 1.3852

Example 3:

Import the x and y components of the Henon system of equations.
Data = ExampleData('henon');
figure('Color', 'k')
plot(Data(:,1), Data(:,2), 'g.')
xlabel('x-component','color','g'), ylabel('y-component','color','g')
set(gca,'color','k'), axis square
Calculate the phase entropy of the y-component in bits (logarithm base 2) without normalization using 7 angular partitions and return the second-order difference plot.
Y = Data(:,2);
Phas = PhasEn(Y, 'K', 7, 'Norm', false, 'Logx', 2, 'Plotx', true)
Phas = 2.0193
Calculate the phase entropy of the x-component using 11 angular partitions, a time delay of 2, and return the second-order difference plot.
X = Data(:,1);
Phas = PhasEn(X, 'K', 11, 'tau', 2, 'Plotx', true)
Phas = 0.8395
