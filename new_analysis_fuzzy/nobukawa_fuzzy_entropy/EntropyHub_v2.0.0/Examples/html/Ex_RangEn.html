Range Entropy: RangEn()

Table of Contents
Syntax: References: Note: Example 1: Example 2: Example 3: Example 4:

Syntax:

[Rangx, A, B] = RangEn(Sig, 'm', 2, 'tau', 1, 'r', 0.2, 'Methodx', 'SampEn', 'Logx', exp(1))
  • Sig Time series signal, a vector of length > 10.
  • m Embedding dimension, a positive integer.
  • tau Time delay, a positive integer.
  • r Distance threshold , a positive scalar.
  • Methodx Base entropy method, either "SampEn" or "ApEn"
  • Logx Logarithm base in Shannon's entropy formula, a positive scalar.
  • Rangx Range entropy estimate
  • A Number of vector matches for embedding dimension (m+1)
  • B Number of vector matches for embedding dimension (m)

References:

  1. Omidvarnia, Amir, et al., Range entropy: A bridge between signal complexity and self-similarity, Entropy 20.12 (2018): 962.
  2. Joshua S Richman and J. Randall Moorman, Physiological time-series analysis using approximate entropy and sample entropy, American Journal of Physiology-Heart and Circulatory Physiology (2000)

Note:

In the following examples, signals / data are imported into MatLab using the ExampleData function from EntropyHub.
To use this function as outlined in the examples below, an internet connection is required.
ExampleData accepts any of the following strings:
  • 'uniform' uniformly distributed random numbers in range [0 1]
  • 'gaussian' normally distributed random numbers with mean = 0, and standard deviation = 1
  • 'randintegers' uniformly distributed pseudorandom integers in range [1 8]
  • 'chirp' chirp signal with the following parameters: f0 = .01, t1 = 4000, f1 = .025
  • 'lorenz' X, Y, Z components of the Lorenz system (sigma: 10, beta: 8/3, rho: 28) [Xo = 10, Yo = 20, Zo = 10]
  • 'henon' X, Y components of the Henon attractor (alpha: 1.4, beta: .3) [Xo = 0, Yo = 0]

Example 1:

Import a sequence of uniformly distributed random numbers in the range [0,1] and calculate the range entropy with sample entropy as the base method and all other default parameters.
X = ExampleData('uniform');
Rangx = RangEn(X)
Rangx = 1.4382

Example 2:

Import a signal of normally distributed random numbers , and calculate the range entropy for using approximate entropy as the base entropy estimator method.
X = ExampleData('gaussian');
Rangx = RangEn(X, 'Methodx', 'ApEn')
Rangx = 1.4897

Example 3:

Import a signal of pseudorandom integers in the range [1, 8] and calculate the range entropy for an embedding dimension (m) of 4, time delay (tau) of 3, a template matching threshold distance of .4 and sample entropy as the base entropy estimator method.
X = ExampleData('randintegers');
Rangx = RangEn(X, 'm', 4, 'tau', 3, 'r', 0.4, 'Methodx', 'SampEn')
Rangx = 0.7585

Example 4:

Import and plot a chirp signal.
Data = ExampleData('chirp');
figure('Color', 'k')
plot(Data, 'g')
